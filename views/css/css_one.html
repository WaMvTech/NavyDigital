    Selecting a Text Editor

    CSS exists for one reason and one reason only: to make your pages look all nice and pretty. CSS, standing for Cascading Style Sheets, is how you tell your browser how to display and style the page. This includes how to layout the page, the color of text, how big the font size is, and much more. Without CSS, the browser would just render HTML documents with the default styles on, but with CSS, we can create magic. Like with our HTML class, you will need a text editor to write and edit your CSS files in. While any editor works, we recommend using the Atom text editor. It will run on the operating system you are currently running on, and the syntax highlighting makes writing CSS really painless.
