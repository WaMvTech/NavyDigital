    Header and Footer tags

    Both tags are pretty straightforward. The header tag defines the header of an article or section, while the footer tag defines the footer.

    Asides

    If you find yourself wanting to go into a bit of detail about the topic at hand, like for example a quick little snippet, use the aside tag.

    Navigation

    To more semantically define where your user can find navigation on your site, take advantage of the nav tag. There's not much to them, just surround the links used for navigation with this tag and you're golden.
